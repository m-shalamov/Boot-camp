Первая строка- import b, поэтому она посетит модуль b
Первая строка в модуле b - import a, поэтому она посетит модуль a
Первая строка в модуле а - это import b , но обратите внимание , что эта строка больше не будет выполняться , потому что каждый файл в python выполняет строку импорта только один раз, и не имеет значения, где и когда она будет выполнена. таким образом, он перейдет к следующей строке и выведет "This is from module a" .
После завершения посещения всего модуля a из модуля b мы все еще находимся в модуле b. поэтому следующая строка будет напечатана "This is from module b"
Строки модуля в выполняются полностью. Итак, мы вернемся к модулю а, где мы начали модуль б.
Импорт строки b уже был выполнен и больше не будет выполнен. в следующей строке будет выведено "This is from module a" , и программа будет завершена.
